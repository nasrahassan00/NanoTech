# Nanotech Research Page
A simple, clean landing page built to practice HTML and CSS fundamentals.

Live Demo: https://nano-tech-page.vercel.app/ 

# About:
This is Project 1 of 8 in my web development learning series. The goal was to focus purely on HTML structure, CSS layout, and basic UI styling, (no JavaScript).


# Features:
1. Desktop-only design
2. Hero section with project overview
3. Informational content about nanotechnology
4. Embedded video (currently desktop only)
5. Clean, minimal visual design


# What I Learned:
1. Semantic HTML 
2. Flexbox & layout spacing
3. Styling typography and containers
4. Designing a simple landing page


# Potential Improvements:
1. Add mobile & tablet responsiveness
2. Fix the video aspect-ratio issue
3. Improve accessibility and color contrast
