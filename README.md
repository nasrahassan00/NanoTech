# Nanotech Research Page
A simple, clean landing page built to practice HTML and CSS fundamentals.

Live Demo: https://nano-tech-page.vercel.app/ 

## About
This is Project 1 of 8 in my web development learning series. The goal was to focus purely on HTML structure, CSS layout, and basic UI styling (no JavaScript).


## Features
- Desktop-only design
  
- Hero section with project overview
  
- Informational content about nanotechnology
  
- Embedded video (currently desktop only)
  
- Clean, minimal visual design


## What I Learned
- Semantic HTML
  
- Flexbox & layout spacing
  
- Styling typography and containers
  
- Designing a simple landing page


## Potential Improvements
- Add mobile & tablet responsiveness
  
- Fix the video aspect-ratio issue
